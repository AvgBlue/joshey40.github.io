Info

Name:

You can either type a name or upload an image for the name. If you type a name, you can also change the font, color and outline color. The zoom slider allows you to adjust the size of the name.

Cost, Power and Description:

These fields allow you to set the cost, power and description of the card. The description should be up to 5 lines long. Line breaks must be done manually.

Image and Frame Break:

You can upload an image for the card and optionally a frame break image. The frame break image should only include those parts of the original image that should be visible outside the frame. I recommend using a tool like remove.bg to remove the unwanted parts of the image.

Frame and Flare:

Here you can select a frame and a flare for the card. More frames and flares will be added in the future, as well as different finishes. If you want to use the round borders for a spell card, just leave the Power field empty (currently only the Basic and Matte frames have a spell variant).
